IoT device simulator
pyIoTDevSim Documentation

pyIoTDevSim is a IoT device simulator that uses a real LoRaWAN® module (connected to serial port) to sent simulated IoT data to TTN® (The Things Network) infrastructure.

A complete features list is found here and the configuration file it is detailed here.
The following LoRaWAN modules are supported:

  • RHF76-052 (since v0.1.0)
  • RHF0M003 (since v0.2.0)
